PlayerPrefs(Preference)를 활용한 데이터 저장
함수 종류
DeleteAll : 모든 키값을 삭제
DeleteKey : 특정 키를 삭제
GetFloat("키(string)", Value); : 같은 타입의 키값을 로드 (예약)한다.
Getint("키(string)", Value);
Getstring("키(string)", Value);
HasKey : 해당 키가 존재하는 지
Save : 변경된 키값을 물리적인 저장공간에 저장한다.
SetFloat("키(string)", Value); : 지정한 키로 같은 타입의 값을 저장한다.
Setint("키(string)", Value);
Setstring("키(string)", Value);
치명적인 단점은 보안성이 제로 이다.
네트워크 게임에서 쓰려면 암호화 해서 하여야 한다.

바이너리(Binary) 파일 저장
이진데이터
데이터를 저장하는 방식중 고전적이지만 물리적인 파일에 데이터를 기록하는 방법을 사용해보자
데이터를 바이너리 파일에 저장하고 필요한 시점에 로드해 사용할 수 있다.


객체 직렬화(Serialization)란?
객체를 통신이나 파일 또는 메모리에 저장 하기 위해 일련의 바이트로 변환 하는것을 말한다.
메모리상에 흩어져 저장된 데이터를 일렬로 재배열해 전송하기에 용이한 데이터로 바꾸는 것을 말한다.

객체 역질렬화(DeSerialization)란?
하드디스크에 저장된 혹은 서버에 저장된 데이터를 다시 실시간으로 데이터 정보를 알려주기 위해 byte를 다시 float이나 int, string 값으로
다시 바꾸어 주는 것을 말한다.
